# myPortfolio

Portfolio Website (React + Tailwind) This project is a personal portfolio website built using React and Tailwind CSS. It serves as a professional introduction to my work, my skills, and my experience as a software development student.

Overview The website is designed to be simple, modern, and fully responsive. It showcases my background, the technologies I work with, and several featured projects I have completed. The goal of this portfolio is to present my capabilities as a developer and designer through clean UI, structured code, and a smooth user experience.

Features Responsive design for mobile, tablet, and desktop Component-based structure using React Section navigation (Home, About, Skills, Projects, Contact) Reusable UI components Tailwind CSS for fast and efficient styling Featured projects displayed with images Contact button that directs users to the Contact section
